My Projects

Wildlife Guardian Alert System

WGAS is a machine learning-based system that uses thermal cameras to detect animals on roads and provide real-time alerts to drivers. This system enhances road safety in Botswana by reducing collisions with wildlife and livestock, thereby protecting lives and supporting the local economy

Grapes Grading System Model

This is a model built to classify grapes into export, local, and drying quality using decision trees, achieving 98% accuracy. This project aims to streamline the grading process in the agricultural industry, enhancing quality control and efficiency.

Autism Prediction Model

This model was created using Random Forest and ensemble learning to assist in the early diagnosis of autism, achieving an impressive 99% accuracy. The project provides healthcare professionals with a reliable AI-driven tool for identifying autism in children, thereby supporting improved early intervention and outcomes.
